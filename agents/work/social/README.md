# agents/work/social/

Platform-adapted public posts generated by the CommunityManager role and
social AutoAgents.

Intermediate planning cards, failed fragments, rejected drafts, and poisoned
repair artifacts are not durable authority merely because they exist in this
folder. Use them only when a current run contract explicitly promotes them.

When the user declares an artifact family poisoned, agents may locate matching
files by filename/path for removal or quarantine, but must not read those files
for requirements, examples, voice, or acceptance criteria.

For S-005, future drafting is blocked until a filename/path-only sweep confirms
old S-005 artifacts across active repo context have been removed, quarantined
outside active drafting context, or explicitly excluded from drafting authority.

## Canonical plan

- [`plan.md`](plan.md) — maintained by `SocialMediaPlanner`, reviewed by
  `CommunityManager`

## Naming convention

```text
YYYY-MM-DD-<channel>-<slug>.md
```

Examples: `2026-04-20-linkedin-bootstrap-launch.md`, `2026-04-25-github-sprint1-recap.md`
