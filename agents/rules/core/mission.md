# Mission

## Goal

Rebuild a public technical profile through visible, agentic engineering work.
Convert scattered real capability into recruiter-readable, peer-inspectable proof.

## Scope

- Public GitHub repository as operational source of truth
- Curated website as public mirror
- LinkedIn / social as campaign distribution layer
- Formal engineering documentation where warranted

## Success criteria

- Portfolio of visible, documented engineering artifacts
- Coherent technical narrative across repo, site, and social
- Demonstrated agentic workflow design as a skill in itself
- Recruiter-facing landing page with AI-readable metadata
- At least one formal case study of the agentic system

## Non-goals

- Building a startup or product
- Becoming an AI influencer
- Mass-producing generic content
- Replacing human judgment with automation
