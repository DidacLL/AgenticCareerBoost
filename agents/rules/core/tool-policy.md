# Tool policy

## Approved toolchain

| Tool | Purpose | Tier |
|---|---|---|
| GitHub | Repository, issues, projects | Free |
| GitHub Pages | Public site hosting | Free |
| GitHub Actions | CI/CD automation | Free tier |
| Jekyll | Static site generator | Free / OSS |
| markdownlint-cli2 | Markdown linting | Free / OSS |
| lychee | Link checking | Free / OSS |
| LaTeX (TeX Live) | Formal documents | Free / OSS |
| latexmk | LaTeX build automation | Free / OSS (ships with TeX Live) |
| xu-cheng/latex-action | GitHub Actions TeX Live runner | Free / OSS |
| Any capable LLM | Agentic execution | Model-agnostic |

## Adding a new tool

Any new tool requires:

1. A pull request with justification
2. Confirmation it has a free or student-accessible tier
3. No vendor lock-in on canonical files
4. Entry added to this table once approved
