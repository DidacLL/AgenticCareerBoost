What this page shows

These projects are here because each one shows a part of the engineering route: agent-assisted workflow control, document tooling, and backend/domain work.

The useful evidence is in the files: source code, static pages, reports, diagrams, state documents, and the decisions left visible enough to review.

The order is intentional. AgenticCareerBoost is the current flagship. P3CTeX shows document/tooling depth. IronBank remains as older backend evidence with banking-domain context.
