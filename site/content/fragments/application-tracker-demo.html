LOCAL TRACKER / READ-ONLY

This is not a separate sample dashboard. The Application Tracker uses the same local dashboard and the same local data store; the read-only wrapper removes only the Raw intake/write surface.

The public site cannot read the local tracker store. For local use, open the tracker with the double-click launchers in the repository.

01 / FULL LOCAL MODE

Double-click application-tracker/Open Application Tracker.cmd to open the full local tracker with Raw intake enabled.

02 / READ-ONLY LOCAL MODE

Double-click application-tracker/Open Application Tracker Read Only.cmd to open the same tracker dashboard without Raw intake/write controls.

03 / DEPLOYED SITE

The deployed site only documents the tracker and points to the local launch flow. It does not load private tracker data and does not expose write controls.
