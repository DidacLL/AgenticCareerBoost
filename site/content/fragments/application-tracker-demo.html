LOCAL TRACKER / READ-ONLY

This is not a separate sample dashboard. The Application Tracker uses the same local dashboard and the same local data store; the read-only wrapper removes only the Raw intake/write surface.

The public site cannot read the local tracker store. Start the local read-only wrapper, then open the URL printed by the command.

01 / START

python application-tracker/acb_tracker_readonly.py

02 / OPEN

Open the read-only tracker URL printed by the local command.

03 / BOUNDARY

No Raw intake, save, or write controls are exposed on the public site.
