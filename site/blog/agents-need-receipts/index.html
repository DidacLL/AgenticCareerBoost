NOTE

Generated work is cheap

Assisted engineering work is easy to produce. That is exactly why it needs friction in the right places.

If a tool changes a project, I want the receipts: files touched, assumptions made, checks run, and the point where a human accepted the change.

Bounded changes. Visible notes. Validation before merge. Human approval before public output.

The rest is theatre with a progress bar.
