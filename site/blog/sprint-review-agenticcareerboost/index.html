AgenticCareerBoost sprint review

What changed when the site stopped being a project subpage and became my main public room.

Home / Blog / Sprint review

REVIEW

The short human version

The recent cleanup was not glamorous. It was mostly URL policy, duplicated state, stale reports, public-site routing, and removing student-first framing from places where it was doing more damage than context.

That kind of work matters because semantic drift is how a public profile starts lying by accident. The README says one thing, the site says another, the CV says a third, and suddenly the system has three identities and no spine. Very modern. Very useless.

The main public site is now didacll.github.io. AgenticCareerBoost is still the source project and proof repository, but it should not be the whole public address. The project is important; the person is not a subfolder.

WHAT CHANGED

Current sprint outcome

url

Root site clarified

The main site moved to the clean GitHub Pages root. Project references still point to AgenticCareerBoost when the reader needs source, reports, or history.

voice

Student-first framing reduced

Education still exists where it belongs: CV and context. The public surface now leads with engineering work, systems judgment, and inspectable artifacts.

state

Backlog and state cleaned

The project state now records the migration and keeps publication, account metadata, outreach, and applications as human-owned gates.

NEXT

What this enables

The next useful move is not another internal cleanup unless something is actually broken. The site is now good enough to support the first public posts and the first outreach batch.

The campaign should use the site as a calm reference surface: one link for humans, one repo for inspection, and posts that point to specific artifacts instead of asking people to believe adjectives.
