NOTE

Project state

This sprint was mostly cleanup: root URL, public copy, duplicated state, stale reports, and pages that had started to describe the project in different ways.

That kind of drift is easy to miss. The site still loads, the README still looks organized, and the state files still have dates. But after enough changes, each surface starts carrying a slightly different version of the project.

The public site now lives at the GitHub Pages root. AgenticCareerBoost remains the source project behind it: workflow files, reports, state documents, site source, validation scripts, and public artifacts.

The copy still needed correction. Some pages had become too careful, too symmetrical, and too self-explanatory. That is the wrong direction. The site should state the work, link the files, and leave the reader enough room to judge.

The next cleanup should remove the remaining defensive style across the whole public surface.
